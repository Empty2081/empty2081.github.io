数据类型

和其他的大多数语言一样, Python也有数据类型, 需要注意的是Python是一门动态强类型的编程语言.

动态：不使用显示数据类型声明，且确定一个变量的类型由初始化的值确定其类型。

强类型：不允许不同类型相加。例如：整形+字符串会报类型错误。

int( 整型 ), float( 浮点数 ), str( 字符串 ), bool( 布尔值 ), list( 列表 ), dict( 字典 ), tuple( 元组 ). ( 暂不谈set类型 )

以上的数据类型很好理解, 需要注意的是元组类型类似C语言的常数, 一经定义不可修改.
